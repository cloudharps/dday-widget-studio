# Notion D-day Widget

날짜와 디자인을 원하는 대로 설정해 Notion에 바로 넣을 수 있는 D-day 위젯입니다.

최대한 편리하게 만들기 위해 D-Day Studio 편집기를 제공해 빠르고 쉽게 원하는 디자인으로 변경할 수 있게 하였습니다.

## 바로 사용하기

### [D-day 위젯 만들기 →](https://notiondday.pages.dev/)

1. 사이트에서 목표일과 문구를 입력합니다.
2. 글꼴, 크기, 굵기, 간격, 색상 등을 원하는 모습으로 조절합니다.
3. 오른쪽 위의 **링크 복사** 버튼을 누르거나 주소창에 있는 링크를 복사합니다.
4. Notion 페이지에서 `/embed`를 입력하거나 임베드 기능을 사용하여 복사한 링크를 붙여넣습니다.

> 복사한 링크를 보관하면 나중에 같은 설정을 다시 열어 수정할 수 있습니다. 일반 링크도 Notion 안에서는 위젯만 표시됩니다.

큰 글씨를 사용하려면 Notion에서 임베드 블록의 높이도 함께 늘려주세요. 위젯은 주어진 임베드 영역 안에서 잘리지 않도록 자동으로 크기를 맞춥니다.

## 설정할 수 있는 항목

- 목표 날짜
- 제목과 제목 크기
- D-day 아래에 표시할 추가 문구
- 앞 문구, 숫자 뒤 문구, 당일 표시 문구
- 글꼴과 글자 크기
- 글자 굵기와 자간
- 배경색과 글자색
- 투명 배경과 텍스트 그림자
- 텍스트 정렬과 모서리 둥글기
- 임베드 영역 크기에 맞춘 글자와 여백 자동 조절
- 실시간 미리보기 박스의 가로·세로 직접 조절
- 매트 아이보리 라이트 모드와 검정·회색 다크 모드

## 표시 방식

- 목표일 전: `D-10일`
- 목표일 당일: `D-DAY`
- 목표일 후: `D+10일`

날짜 계산은 위젯을 보는 기기의 현지 날짜를 기준으로 자동 갱신됩니다.

## 링크와 임베드의 차이

- 일반 링크로 접속하면 디자인을 수정할 수 있는 편집 화면이 표시됩니다.
- iframe 또는 Notion 임베드 안에서는 설정 UI가 숨겨지고 위젯만 표시됩니다.
- `embed=1`이 포함된 링크는 일반 브라우저에서도 위젯만 표시합니다.

## 글꼴 라이선스

위젯에서 제공하는 Noto, 나눔, 고운, 검은고딕 계열은 Google Fonts에서 배포하는 무료 사용 가능 글꼴입니다. 각 글꼴의 라이선스는 [Google Fonts 저장소](https://github.com/google/fonts)의 해당 글꼴 디렉터리에서 확인할 수 있습니다.
